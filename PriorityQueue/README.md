# C++ Minimum Priority Queue
This is a project that creates a minimum priority queue utilizing heaps within C++

# Getting Started
To get started please clone this repository into a dedicated location on your system

``git clone https://github.com/aryah-rao/ARaoAAnguloETecson_cs271_project2``

To compile your code please execute:

``make``

To test minqueue.cpp and minqueue.h please execute:

``./test``

To test usecase.cpp and main.cpp please execute:

``./usecase``
